Shells

Server​

Require​

Require shell functions from lib.

local shell = require '@wc_lib.lib.shells.server.sv_main'

shell.new​

Creates a new shell.

local shell = shell.new(source, data)

Example:
local shell = shell.new(source, {
    model = 'shell_model',
    coords = coords,
    enterOffset = ???,
    bucket = 2
})

shell.get​

Get the shell.

local shell = shell.get(data)

shell.triggerClientEvent​

Triggers client event for everyone who's in the shell.

shell:triggerClientEvent(event, ...)

Example:
shell:triggerClientEvent('example:example', data)

shell.enter​

Enter shell.

shell:enter(source)

shell.exit​

Exit shell.

shell:exit(source)

shell.addProp​

Add prop to shell (such as furniture etc).

local success, entity = shell:addProp(data)

Example: 
local success, entity = shell:addProp({
    model = 'model',
    coords = coords,
    rotation = rotation
})

shell.removeProp​

Removes prop from the shell.

local success = shell:removeProp(entity)
